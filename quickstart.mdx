## Prerequisites

- A Brain API account ([sign up free](https://brainapi.dev/dashboard))
- An API key from at least one LLM provider (Anthropic, OpenAI, or Google)

---

## Step 1: Connect an AI Provider

<Steps>
  <Step title="Go to Integrations">
    Navigate to **Dashboard → Integrations** and find your LLM provider (Anthropic, OpenAI, or Google).
  </Step>
  <Step title="Add your API key">
    Click **Connect**, paste your API key, and click **Save**.

    Your key is encrypted with AES-256-GCM and never stored in plaintext.
  </Step>
  <Step title="Verify connection">
    A green **Connected** badge appears when your key is valid.
  </Step>
</Steps>

---

## Step 2: Chat with Your First Agent

<Steps>
  <Step title="Open Agent Chat">
    Go to **Dashboard → Agent**. You'll see the chat interface.
  </Step>
  <Step title="Send a message">
    Type anything — a question, a task, or a command:
    ```
    Summarize the key differences between REST and GraphQL APIs.
    ```
  </Step>
  <Step title="Watch it respond">
    Your agent responds using the connected LLM. Token usage is shown in real-time at the top of the page.
  </Step>
</Steps>

---

## Step 3: Connect an Integration (MCP)

<Steps>
  <Step title="Go to Integrations">
    Navigate to **Dashboard → Integrations**.
  </Step>
  <Step title="Connect Slack (or any other)">
    Click **Connect** on Slack, paste your Bot Token, and save.

    <Note>
      For OAuth-based integrations (GitHub, Google), click **Connect with OAuth** and authorize in the popup.
    </Note>
  </Step>
  <Step title="Use it in chat">
    Ask your agent to use it:
    ```
    Send a message to the #general Slack channel: "Brain API is live! 🚀"
    ```
    The agent automatically calls the Slack MCP tool.
  </Step>
</Steps>

---

## Step 4: Create a Workflow

<Steps>
  <Step title="Go to Workflows">
    Navigate to **Dashboard → Workflows** → **New Workflow**.
  </Step>
  <Step title="Add nodes">
    Drag nodes from the panel onto the canvas:
    1. **Trigger** node — set to Manual for now
    2. **LLM Call** node — connect it to the Trigger
    3. **MCP Tool** node — connect it to the LLM Call
  </Step>
  <Step title="Configure the LLM node">
    Click the LLM node and set:
    - **Prompt**: `Summarize this text: {{trigger.input}}`
    - **Model**: Claude Sonnet 4.6
  </Step>
  <Step title="Run it">
    Click the **Run** button at the top. Results appear in the panel on the right.
  </Step>
</Steps>

---

## Step 5: Automate with a Recipe

<Steps>
  <Step title="Go to Recipes">
    Navigate to **Dashboard → Recipes** → **New Recipe** or pick a template.
  </Step>
  <Step title="Choose a trigger">
    - **Webhook**: fires when an external system POSTs to your unique URL
    - **Schedule**: cron expression (e.g., `0 9 * * 1` = every Monday 9am)
  </Step>
  <Step title="Trigger via webhook">
    ```bash
    curl -X POST https://brainapi.dev/api/automations/trigger \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"input": "What is the weather like today?"}'
    ```
  </Step>
</Steps>
