## February 2026

### v0.9.0 — A2A Protocol (2026-02-27)

**Brain API is now an A2A-native platform.**

- `/.well-known/agent.json` — Brain API's Agent Card is publicly discoverable
- `/api/a2a/tasks` — Accept tasks from any A2A-compatible external agent
- `/api/a2a/agents/discover` — Discover and connect external A2A agents
- `/api/a2a/agents/[id]/delegate` — Delegate tasks to external agents
- **Dashboard → A2A Protocol** — 3-tab UI: Agent Card, Connected Agents, Task History
- Full A2A type system (`src/types/a2a.ts`)
