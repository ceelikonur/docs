## How Integrations Work

Every Brain API integration is a **MCP Server** — a typed, testable connection to an external service. Your agent automatically discovers available tools and calls them when needed.

See [MCP Integrations](/fundamentals/mcp-integrations) for the full architecture overview.

## Build Your Own

Use the MCP scaffold to create a custom integration:

```bash
node scripts/scaffold-mcp.mjs my-service
```

Then host it on Brain API's cloud or run it locally. See [MCP Development Guide](/fundamentals/mcp-integrations#building-a-custom-mcp-server).
